# 📂 JUNIOR SYSTEM ADMINISTRATOR PROTOCOL — BEGINNER TRACK

## 👨‍👦 Note for Mission Command (Dad)

This track uses three tricks from learning science:

1. **Warm-Up Drills** start every module. They ask the cadet to redo one skill from an *earlier* module **from memory, without looking anything up**. Struggling for a minute is the point — that struggle is what makes the memory stick. If he's stuck after a real try, let him look back at the old module, close it, and try again from memory.
2. **On Your Own** tasks end every module. They give a goal but **no recipe**. Earlier steps hand him the exact commands; this step checks whether he can produce them himself.
3. **Debriefs** ask him to explain an idea back to you in his own words. If he can't explain it, he memorized the steps but missed the idea — just talk it through together and move on. No grades.

Pace: one module per session, and don't rush the track. A few days between modules actually *helps* memory. The final module is a field test covering everything — he should use the hints at the end of it freely; they point back to the module where the skill lives.
